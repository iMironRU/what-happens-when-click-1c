# Гайд для авторов

## Структура репозитория

Каждый **параграф** — отдельный `.md`-файл в папке соответствующего модуля.  
Каждый **модуль (часть)** — папка `chapters/NN_slug/`.

```
chapters/
  NN_slug/
    NN-01_slug.md   ← первый файл: начинается с # Часть N. Название
    NN-02_slug.md   ← последующие: начинаются с ## N.2. Название
    NN-MM_itog.md   ← последний файл модуля
```

## Именование

- Папки: `NN_slug` — двузначный номер + дефисный латинский slug
- Файлы: `NN-MM_slug.md` — номер модуля, номер параграфа, slug
- Последний файл каждого модуля: `NN-MM_itog.md`

## Структура параграфа

### Первый файл модуля (`NN-01_...md`)

Начинается с заголовка модуля:

```markdown
# Часть N. Название

## N.1. Название параграфа

### Открытие
[Связь с предыдущим + ориентир]

### Раздел 1
[Содержание]

> Выноска с ключевой мыслью.

### Главное
[Один плотный абзац — пересказ ключевых мыслей параграфа]

### Мостик
[Переход к следующему параграфу]
```

### Последующие файлы (`NN-02_...md` и далее)

Начинаются сразу с заголовка параграфа:

```markdown
## N.2. Название параграфа

### Открытие
...
```

### Файл итога (`NN-MM_itog.md`)

```markdown
## Итог Части N

[Краткое резюме: что изучили, какие понятия ввели]

**В следующей части** — ...
```

## Уровни заголовков (важно для Pandoc)

- `#` — заголовок модуля (только в первом файле)
- `##` — заголовок параграфа
- `###` — подраздел внутри параграфа
- `####` — подподраздел (использовать минимально)

**Замечание:** модули 05–09 в текущей редакции используют `##` для модуля и `###` для параграфов.
При следующей ревизии привести к единому стилю (модуль = `#`, параграф = `##`).

## Запреты

- Не менять нумерацию файлов без обновления `docs/plan.md`
- Не добавлять файлы вне `chapters/` (кроме `assets/`, `docs/`, корневых служебных файлов)
- Не использовать абсолютные пути в ссылках на изображения — только `../../assets/img/...`

## Добавление нового модуля

1. Создать папку `chapters/NN_slug/`
2. Создать файлы `NN-01_...md` … `NN-MM_itog.md`
3. Обновить статус в `docs/plan.md`

Сборка запустится автоматически при пуше в `main`.
